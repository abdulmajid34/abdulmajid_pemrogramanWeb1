## JS Class

Class adalah cetak biru atau template untuk membuat objek. Bayangkan class seperti resep kue. Resep berisi daftar bahan dan langkah-langkah, tapi resep itu sendiri bukan kue — ia hanya panduan.

## Instance

Instance adalah objek nyata yang dibuat dari sebuah class. Jika class adalah resep, maka instance adalah kue yang sudah jadi. Kita bisa membuat banyak instance dari satu class, masing-masing dengan data yang berbeda.

## Inheritance

Inheritance adalah kemampuan sebuah class untuk mewarisi properti dan method dari class lain. Ini seperti membuat resep turunan dari resep dasar. Misalnya, dari resep "Kue" dasar, kita buat resep baru "KueLapis" yang memiliki semua bahan dan langkah Kue, ditambah bahan/langkah khusus.
